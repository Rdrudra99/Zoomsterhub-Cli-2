## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/Rdrudra99/Zoomsterhub-Cli-2.git
cd Zoomsterhub-Cli-2
\`\`\`

2. Install dependencies:
\`\`\`bash

\`\`\`bash
